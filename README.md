## 🚀 Featured Live Project

### 🟢 HR Attrition Analyst — AI Application

An AI-powered HR analytics app that turns raw employee data into plain-English attrition insights. The LLM layer (Groq + Llama 3) generates narrative summaries of attrition patterns and risk factors, on top of a full data pipeline. Built end-to-end and deployed on Streamlit.

**🔗 Live Demo:** [hr-attrition-ai-analyst.streamlit.app](https://hr-attrition-ai-analyst.streamlit.app/)

**Stack:** Python · Streamlit · Groq API · Llama 3

## 🏆 Projects

### 🚢 Titanic Survival Prediction *(Working with Categorical Data)*

Full Kaggle-style ML pipeline with categorical feature engineering — title extraction, rare category handling, family size, fare-per-person — evaluated across four classifiers.

**Key techniques:** Missing value imputation · One-hot encoding · Feature extraction from names · Cross-validation · Feature importance analysis

**Models:** Logistic Regression · Random Forest · Gradient Boosting · XGBoost

---

### 🎓 Student Performance Analysis *(Working with Numerical Data)*

End-to-end numerical data project covering EDA, outlier treatment, feature scaling, binning, and polynomial feature generation on a real academic dataset.

**Key techniques:** Normalization · Standardization · Outlier detection · Correlation analysis · Polynomial features

---

### 💰 Salary Prediction

Regression project predicting employee compensation from input features, with feature selection, model training, and evaluation metrics.

**Key techniques:** Feature selection · Regression modeling · Model evaluation

---

### 📧 Spam Classifier

NLP pipeline classifying messages as spam or ham — tokenization, TF-IDF vectorization, and classification model with evaluation.

**Key techniques:** Text preprocessing · Tokenization · TF-IDF · Classification · Model evaluation
